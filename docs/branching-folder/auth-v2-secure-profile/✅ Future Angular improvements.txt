I meant a small UX improvement that big sites like Amazon / Flipkart / Google login pages use.
Right now your flow is:
Login
↓
Forgot Password (separate page)
↓
Enter email
↓
Send reset link

But a more professional UX is to show the Forgot Password inside the login page itself as a modal or toggle.
Example flow used by many apps:

Login Page
 ├─ Email
 ├─ Password
 └─ Forgot Password?
        ↓
   small reset form opens
        ↓
   enter email
        ↓
   send reset link

This avoids navigating to another page.

Example UI improvement

Instead of redirecting to:

/forgot-password

You show a small form:

Forgot your password?

[ Email input ]

[ Send reset link ]

directly inside the login page.
